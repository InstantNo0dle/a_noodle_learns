# Orbital Velocity
This is the speed that enables one object to orbit another object, and depends on the radius of the object.
$$\begin{align}
F_G = ma = m\frac{v^2}{r} \\
\rightarrow G\frac{mM}{r^2} = m\frac{v^2}{r}\\
\therefore v^2 = \frac{GM}{r}\\
v = \sqrt{\frac{GM}{r}}
\end{align}
$$
