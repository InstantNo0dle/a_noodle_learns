# Newton's Second Law
Newton's second law can be used to relate the net [[Torque]] to the [[Angular Acceleration]] it causes about a rotational axis.
$$
\tau_{net} = I\alpha
$$

For [[Angular Momentum]], it can be written as: ^8869df
$$
\vec \tau_{net} = \frac{d\vec l}{dt}
$$
