# Organic nomenclature

Functional groups
- Alkane
	- All carbon and hydrogen
	- All single bonds
	- suffix = "ane"
	- $\ce{C_nH_{2n+2}}$
- Alkene
	- All carbon and hydrogen
	- One double bond
	- suffix = "ene"
	- $\ce{C_nH_{2n}}$
- Alkyne
	- All carbon and hydrogen
	- One triple bond
	- suffix = "yne"
	- $\ce{C_nH_{2n-2}}$
- Phenyl/benzene ring
- Amine
- Alcohol
	- Suffix = "ol"
- Aldehyde
	- Suffix = "al"
- Ketone
	- Suffix = "one"
